A philosophy document.

The Kernel defines the language of the platform.
The Kernel has zero infrastructure dependencies.
Every dependency points downward.
Every public type has explicit invariants.
Runtime executes; Kernel models.
Adapters integrate; they never define domain concepts.
Creative Studio validates every abstraction before it becomes part of the Kernel.

Whenever we're unsure about a future design, we don't ask "what's easier to code?"—we ask "does it respect the Kernel Principles?"
